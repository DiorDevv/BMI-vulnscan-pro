SQL In'ektsiya Skaneri

Kritik xavf

Ma'lumotlar bazasiga ruxsatsiz kirish imkonini beradigan SQL injection zaifliklarini aniqlaydi. Ilova kirish maydonlari, URL parametrlari va cookie'larda maxsus payload'lar sinab ko'riladi.

✓ Error-based SQL injection
✓ Boolean-based blind injection
✓ Time-based blind injection
✓ UNION-based extraction test
🎭

Cross-Site Scripting (XSS)

Yuqori xavf

Brauzerda ixtiyoriy JavaScript kodni bajarish imkonini beradigan XSS zaifliklarini topadi. Foydalanuvchi sessiyasini o'g'irlash, phishing va keylogger hujumlarida ishlatiladi.

✓ Reflected XSS (URL parametrlarida)
✓ Stored XSS (ma'lumotlar bazasida)
✓ DOM-based XSS
✓ 50+ zamonaviy XSS payload'lari
🌐

CORS Noto'g'ri Sozlama

Yuqori xavf

Cross-Origin Resource Sharing (CORS) politikasidagi xatolarni aniqlaydi. Noto'g'ri sozlangan CORS boshqa saytlarga API so'rovlar yuborish imkonini beradi.

✓ Wildcard origin (*) tekshiruvi
✓ Credentials bilan birga wildcard
✓ Reflection attack tekshiruvi
✓ Null origin qabul qilish
🛡️

Xavfsizlik Sarlavhalari

O'rta xavf

HTTP javob sarlavhalarida muhim xavfsizlik sozlamalarining mavjudligini tekshiradi. Etishmayotgan sarlavhalar bir qator hujumlarni osonlashtirishi mumkin.

✓ Strict-Transport-Security (HSTS)
✓ Content-Security-Policy (CSP)
✓ X-Frame-Options (Clickjacking)
✓ X-Content-Type-Options, Referrer-Policy
🔒

SSL/TLS Tahlili

O'rta xavf

HTTPS sertifikati va TLS protokoli konfiguratsiyasini chuqur tekshiradi. Eskirgan protokollar, zaif sertifikatlar va noto'g'ri sozlamalar aniqlanadi.

✓ Sertifikat muddati va ishonchlilik
✓ TLS 1.0/1.1 (eskirgan) aniqlanishi
✓ Zaif cipher suite'lar
✓ Self-signed sertifikat tekshiruvi
📁

Directory Listing & Sezgir Fayllar

O'rta xavf

Veb-serverda ochiq kataloglar va sezgir fayllarni qidiradi. .env, .git, backup, konfiguratsiya fayllarining ochiq qolishi jiddiy xavf hisoblanadi.

✓ Ochiq kataloglar (Apache, Nginx)
✓ .env, .git, .htaccess fayllar
✓ Backup fayllar (.bak, .old, ~)
✓ Admin panel, phpinfo topish

Port Skaneri

✓ Keng tarqalgan portlar tekshiruvi
✓ Xizmat identifikatsiyasi (SSH, FTP, DB)
✓ Ma'muriy portlar (3306, 5432, 6379)
✓ Async parallel skanerlash

Ochiq Yo'naltirish (Open Redirect)

Past xavf

URL parametrlari orqali foydalanuvchini boshqa saytga yo'naltirish imkoniyatini tekshiradi. Phishing va OAuth token o'g'irlash hujumlarida ishlatiladi.

✓ redirect=, url=, next= parametrlar
✓ Encoding bypass test
✓ Protocol-relative URL tekshiruvi
✓ Meta refresh redirect aniqlash
